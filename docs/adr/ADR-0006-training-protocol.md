# ADR-0006: Staged training (Monty design)
Boot camp: XAUUSD, ONE week M1, repeat until PERFECT = +2x goal EVERY day, zero floor touches;
then SECOND unseen week same bar. Very explorative week one. Expansion: more symbols same week
(EURUSD/GBPUSD/US30) -> deep history 2020-2026 all eras -> randomized goal/floor (any-X).
Champion-vs-challenger metric: consistency winning rate = % days goal-hit-no-breach.
Frozen means frozen; ledger/bench/promotions update only inside training sessions.
Weekly retrain reminder to Monty, forever. Runs <= a few hours before review.
